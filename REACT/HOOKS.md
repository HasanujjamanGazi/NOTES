### USE STATE

> `user` is a variable & `setUser` is a function that can change the value of `user` . Inside the `useState()` there goes the initial value of the variable `user`.

> **Increment** operator doesn't work inside the `setUser` function.

> `setUser` is a `async` function.  

```javascript
import { useState } from "react";
const [user, setUser] = useState(__initial_value__);
```
