## DOCKER

```bash
# To check docker installed & version.
docker --version

# HELP
docker

# To download the docker image (only download).
docker pull 

# Creat & run a docker container from image.
docker run

# List all running containers
docker ps

# List all containers
docker ps -a

# List docker images
docker images

# To delete docker container
docker rm name/ID

# To delete docker images
docker rmi name/ID

# To rename a docker container
docker rename name new-name

```
